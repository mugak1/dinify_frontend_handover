Reports
Sales Report (UGX)

This report shows the sales/revenue made by the restaurant over a period of time; that is, daily, weekly, monthly, quarterly and annually. The user is prompted to select a desired period to view the report listing and summary overview. A custom date range may also be selected

Menu Report

The Menu report shows the performance of the menu items, menu groups and menu sections at a restaurant for the selected time period.

Transactions Report (UGX)

This report shows the payment operations of the restaurant. The user views payments received by Dinify on behalf of the restaurant as well as remittances from Dinify to the restaurants. This analysis is done for the selected period, that is, daily, weekly, monthly, quarterly and annually.

Diners Report

This report shows the list of diners using Dinify at that restaurant. The user is prompted to select their period of interest e.g. daily, weekly, monthly. quarterly and annually.
